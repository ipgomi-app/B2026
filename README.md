# ENG · TSPG 부산권역 현장 솔루션 제안 (PWA)

LOGISALL ENGINEERING · TSPG 자동화창고 현장 영업용 단일 페이지 웹앱.

## 구성
- `index.html` — 루트 진입(앱으로 자동 이동)
- `tspg_field_proposal_app.html` — 메인 앱
- `manifest.webmanifest`, `sw.js` — PWA(오프라인 · 홈화면 설치)
- `icon-192.png`, `icon-512.png`, `apple-touch-icon.png` — 앱 아이콘(네이비 CI)

## 배포
GitHub Pages(HTTPS) 권장. 카톡 공유·홈화면 설치 기능은 HTTPS에서 동작합니다.

- 접속 주소: `https://<사용자명>.github.io/<저장소명>/`
